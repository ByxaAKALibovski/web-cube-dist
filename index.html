Взаимодействие с DOM через JavaScript

Введение

Привет, ребята! На этом уроке мы изучаем, как с помощью JavaScript "оживлять" веб-страницы, работая с DOM (Document Object Model). DOM — это как карта страницы, а JavaScript — наш инструмент, чтобы менять её содержимое и поведение.

Теория

Объект document — это главный "ключ" к DOM. С его помощью мы можем находить элементы на странице и изменять их. Вот что мы узнали:

Основные методы и свойства:

  • document.getElementById(id) — находит элемент по его ID.
  • document.querySelector(selector) — находит первый элемент по CSS-селектору.
  • document.querySelectorAll(selector) — находит все элементы по CSS-селектору.
  • element.innerHTML — меняет или получает HTML внутри элемента.
  • element.textContent — меняет или получает текст внутри элемента.
  • element.addEventListener(event, handler) — добавляет реакцию на действия (например, клик).

1. Игра в крестики-нолики

Создай поле 3x3. При клике на ячейку ставь "X" или "O" в зависимости от хода. Попробуй прямо сейчас!

Пример кода:

HTML:

<div id="game-board">
    <div class="cell"></div>
    <div class="cell"></div>
    <div class="cell"></div>
    <div class="cell"></div>
    <div class="cell"></div>
    <div class="cell"></div>
    <div class="cell"></div>
    <div class="cell"></div>
    <div class="cell"></div>
</div>

CSS:

#game-board {
    display: grid;
    grid-template-columns: repeat(3, 100px);
    gap: 5px;
    margin: 20px auto;
    width: 310px;
}

.cell {
    width: 100px;
    height: 100px;
    background-color: #fff;
    border: 2px solid #333;
    display: flex;
    align-items: center;
    justify-content: center;
    font-size: 40px;
    cursor: pointer;
}

JavaScript:

const cells = document.querySelectorAll('.cell');
let currentPlayer = 'X';

cells.forEach(cell => {
    cell.addEventListener('click', () => {
        if (cell.textContent === '') {
            cell.textContent = currentPlayer;
            currentPlayer = currentPlayer === 'X' ? 'O' : 'X';
        }
    });
});

2. Калькулятор цвета

Создай кнопку, которая при клике меняет цвет фона всей страницы на случайный. Нажми и убедись!

Пример кода:

HTML:

<button id="color-btn">Сменить цвет</button>

JavaScript:

document.getElementById('color-btn').addEventListener('click', () => {
    document.body.style.backgroundColor = '#' + Math.floor(Math.random()*16777215).toString(16);
});

3. Список желаний

Создай поле ввода и кнопку. При нажатии на кнопку текст из поля добавляется в список ниже. Введи что-нибудь!

Пример кода:

HTML:

<input type="text" id="wish-input" placeholder="Введи желание">
<button id="add-wish">Добавить</button>
<ul id="wish-list"></ul>

JavaScript:

document.getElementById('add-wish').addEventListener('click', () => {
    const input = document.getElementById('wish-input');
    const wish = input.value;
    if (wish) {
        const li = document.createElement('li');
        li.textContent = wish;
        document.getElementById('wish-list').appendChild(li);
        input.value = '';
    }
});

    Ваша задача — создать веб-страницу, которая меняет цвет фона параграфа при нажатии на кнопку. Это задание поможет вам закрепить навыки работы с DOM и событиями в JavaScript. Выполните все шаги точно так, как указано ниже.

    Условия выполнения:

    1. Создайте HTML-страницу:
      • Добавьте кнопку с id="change-color" и текстом внутри: "Изменить цвет".
      • Добавьте параграф с id="text" и текстом внутри: "Этот текст будет менять цвет фона".
      • Ничего лишнего добавлять не нужно.
    2. Добавьте стили:
      • В секции <style> внутри <head> задайте следующие правила:
      • Для элемента с id="text" установите padding: 20px; и border: 1px solid #ccc;.
      • Другие стили добавлять не нужно.
    3. Напишите JavaScript-код:
      • Используйте метод document.getElementById для получения элементов кнопки и параграфа.
      • Добавьте обработчик события клика на кнопку с помощью addEventListener.
      • При нажатии на кнопку цвет фона параграфа должен измениться на случайный цвет.
      • Для генерации случайного цвета используйте формулу: '#' + Math.floor(Math.random()*16777215).toString(16).

    Итоговый код:

    Ваш код должен выглядеть ровно так, как показано ниже. Не добавляйте никаких изменений или дополнительных функций.

    <!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Домашнее задание</title>
    <style>
        #text {
            padding: 20px;
            border: 1px solid #ccc;
        }
    </style>
</head>
<body>
    <button id="change-color">Изменить цвет</button>
    <p id="text">Этот текст будет менять цвет фона</p>
    <script>
        document.getElementById('change-color').addEventListener('click', function() {
            const randomColor = '#' + Math.floor(Math.random()*16777215).toString(16);
            document.getElementById('text').style.backgroundColor = randomColor;
        });
    </script>
</body>
</html>
